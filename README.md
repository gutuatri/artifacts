# Gutuatri Artifacts
Free, open-source AI tools for people the hiring system wasn't built for.
These artifacts run inside Claude.ai using your existing Claude subscription. No API key. No setup. No cost beyond what you already pay.

What's here
Resume Creation Hub
A five-tab tool that builds a tailored resume, cover letter, outreach email, and follow-up email from your existing documents and a job posting.
What it does:

Extracts your full career profile from uploaded resumes, cover letters, bios, and LinkedIn exports (PDF, DOCX, images)
Detects gaps between your profile and the target job
Walks you through a structured gap interview in plain language — no resume-speak required
Generates a tailored resume using proven frameworks (Blend, Jenny Foss, Laszlo Bock, Austin Belcak)
Runs an honesty audit that flags any tool or skill not in your actual profile
Scores the resume with a detailed scorecard and fix buttons
Downloads as .docx

File: resume-creation-hub.html

How to use it

Go to claude.ai and sign in (free account works)
Start a new conversation
Copy the contents of the artifact file
Paste it into the chat and hit send
Claude will render the tool — use it directly in the chat window

That's it. No installation. No accounts. No API keys.

What it costs
Nothing extra if you have a Claude.ai account. The artifact makes API calls that are absorbed by your subscription. Free tier accounts have message limits — if you hit them, wait for the limit to reset or upgrade to Claude Pro.

Who built this
